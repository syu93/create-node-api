Create node API is a simple [npm initializer](https://docs.npmjs.com/cli/init) to bootstrap a simple yet well organised Node.js API using [Express](https://expressjs.com).

Folder structure :

```
-- actions/
	-- app.js
	-- index.js
-- boot/
	-- index.js
-- config/
	-- config.json.example
	-- index.js
-- helpers/
	-- index.js
-- middlewares/
	-- ensureFields.js
	-- index.js
	-- res.js
-- models/
	-- index.js
-- routes/
	-- app.js
	-- index.js
-- index.js
```
